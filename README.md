### Build/Run

#### Requirements

- Node.js
- Yarn

```javascript

/* First, Install the needed packages */
yarn

/* Then start the React app */
yarn dev

/* Then start web  */
yarn dev:web

/* Then start mock server  */
yarn dev:api

```
